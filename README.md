# 👥 Exercício 03 — Idade Média entre Duas Pessoas

Programa de estrutura sequencial que lê o nome e a idade de duas pessoas e exibe uma mensagem com a **idade média** entre elas, com uma casa decimal.

---

## 🧠 Conceitos Aplicados

- Estrutura sequencial
- Tipo `string`
- Leitura de mais de um dado na mesma linha com `Split`
- Casting de tipos
- Formatação de saída com casas decimais

---

## 📥 Entrada

Duas linhas, cada uma contendo o **nome** e a **idade** separados por espaço:

| Linha | Formato        | Exemplo     |
|-------|----------------|-------------|
| 1     | `nome1 idade1` | `Maria 19`  |
| 2     | `nome2 idade2` | `Joaquim 20`|

## 📤 Saída

Mensagem com os nomes e a idade média calculada:

```
A idade média de <nome1> e <nome2> é de <media> anos
```

---

## 💡 Exemplo

**Entrada:**
```
Maria 19
Joaquim 20
```

**Saída:**
```
A idade média de Maria e Joaquim é de 19.5 anos
```
